# Jam2 Blog 個人部落格網站
> [Demo 連結] (https://is-jam.vercel.app/#/)
## 網站功能
1. 部落格文章：分類檢索、搜尋
2. 後台管理員登入
3. 數據儀錶板
4. 文章管理
## 使用技術
1. `Vue 3 Composition API` JS 框架
2. `Tailwind CSS` 框架  + `Post CSS ` CSS後處理器
3. `Vue Router` 前端模擬路由
4. `Pinia` 狀態管理工具
5. `Axios` API 串接
6. `Unhead` SEO 優化，新增 head meta 資訊
7. `TinyMCE` HTML編輯器
8. `highlight.js` 美化程式碼區塊套件
9. `chart.js` 數據視覺化圖表套件
10. `VITE`建構工具 + `Vercel` 自動部署

## 其他
1. 埋入 GTAG：追蹤網頁流量
